# 개발 단계

개발 단계에는 다음 활동이 포함됩니다.

- 로컬 및 스테이징 환경 설정
- 스프린트 계획
- 티켓 실행
- 문제 해결
- 코드 검토, 병합 및 테스트
- 스프린트 검토
- 고객 승인

>[!TIP]
>
>다음을 참조하십시오 [일반 모범 사례](general.md) 개발 프로세스의 전반적인 관리에 대한 높은 수준의 권장 사항.

다음 섹션에는 개발 단계에 대한 모범 사례 정보가 포함되어 있습니다.
