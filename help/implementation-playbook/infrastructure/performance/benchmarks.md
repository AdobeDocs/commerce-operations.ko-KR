# 벤치마크

추가 사용자 지정 및 CDN 캐시가 없는 클라우드 인프라의 Adobe Commerce 2.4.5에는 다음과 같은 벤치마크 수치가 있습니다.

- 2억 4,200만 개의 유효 SKU
- 300ms 평균 API 응답 시간
- 시간당 208K 주문
- &lt;2s 평균 상점 응답 시간
- 시간당 200만 페이지 보기 수
