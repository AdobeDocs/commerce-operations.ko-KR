# ACP2E-4395: 기프트 카드 제품은 글로벌 범위의 금액 없이 API를 통해 생성 할 수 있습니다

ACP2E-4395 패치는 글로벌 범위의 금액 없이 API를 통해 기프트 카드 제품이 생성되는 문제를 수정합니다. 이 패치는 [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.81이 설치된 경우에 사용할 수 있습니다. 패치 ID는 ACP2E-4395입니다. 이 문제는 Adobe Commerce 2.4.9에서 수정됩니다.

## 영향을 받는 제품 및 버전

**Adobe Commerce 버전에 대한 패치가 만들어졌습니다.**

* Adobe Commerce(모든 배포 방법) 2.4.7-p8

**Adobe Commerce 버전과 호환:**

* Adobe Commerce(모든 배포 방법) 2.4.5 - 2.4.8-p5

>[!NOTE]
>
>새 [!DNL Quality Patches Tool] 릴리스가 있는 다른 버전에 패치를 적용할 수 있습니다. 패치가 Adobe Commerce 버전과 호환되는지 확인하려면 `magento/quality-patches` 패키지를 최신 버전으로 업데이트하고 [[!DNL Quality Patches Tool]에서 호환성을 확인합니다. 패치 검색 페이지](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). 패치 ID를 검색 키워드로 사용하여 패치를 찾습니다.

## 문제

기프트 카드 상품은 글로벌 범위의 금액 없이 API를 통해 생성 할 수 있습니다.

<u>재현 단계</u>:

1. 다음 단계에서 사용할 기프트 카드 금액 속성 ID를 가져옵니다.

   ```
   GET /rest/V1/products/attributes/giftcard_amounts
   ```

1. API를 통해 *모든 웹 사이트*&#x200B;에 대한 금액을 제외한 웹 사이트 범위에서만 기프트 카드 제품을 만드십시오(`website_id` = `0`).

   ```
   POST /rest/V1/products
   {
       "product": {
           "sku": "test-giftcard-bug",
           "name": "Test Gift Card",
           "attribute_set_id": 4,
           "status": 1,
           "visibility": 4,
           "type_id": "giftcard",
           "price": 0,
           "extension_attributes": {
               "website_ids": [1],
               "stock_item": {
                   "qty": 1000,
                   "is_in_stock": true,
                   "manage_stock": false,
                   "use_config_manage_stock": false
               },
               "giftcard_amounts": [
                   { "website_id": 1, "value": 50, "attribute_id": <attribute_id> }
               ]
           },
           "custom_attributes": [
               { "attribute_code": "giftcard_type", "value": "0" },
               { "attribute_code": "allow_open_amount", "value": "0" },
               { "attribute_code": "is_redeemable", "value": "1" }
           ]
       }
   }
   ```

<u>예상 결과</u>:

요청이 HTTP 400을 사용하여 거부되었으며 메시지: *모든 웹 사이트 범위에 대한 금액을 지정하거나 금액을 열어야 합니다*.

<u>실제 결과</u>:

글로벌 *모든 웹 사이트* 범위에 금액이 없더라도 요청이 HTTP 200을 반환하고 기프트 카드 제품을 만들어 잘못된 상태로 둡니다.
