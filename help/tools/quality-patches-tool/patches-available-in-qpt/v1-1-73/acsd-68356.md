# ACSD-68356: [!DNL GraphQL] 장바구니 쿼리가 가상 견적에 대해 잘못된 할인 금액을 반환합니다.

ACSD-68356 패치는 [!DNL GraphQL] 장바구니 쿼리가 가상 견적에 대해 잘못된 할인 금액을 반환하는 문제를 해결합니다. 이 패치는 [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73이 설치된 경우에 사용할 수 있습니다. 패치 ID는 ACSD-68356입니다. 이 문제는 Adobe Commerce 2.4.9에서 수정됩니다.

## 영향을 받는 제품 및 버전

**Adobe Commerce 버전에 대한 패치가 만들어졌습니다.**

* Adobe Commerce(모든 배포 방법) 2.4.8-p2

**Adobe Commerce 버전과 호환:**

* Adobe Commerce(모든 배포 방법) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>새 [!DNL Quality Patches Tool] 릴리스가 있는 다른 버전에 패치를 적용할 수 있습니다. 패치가 Adobe Commerce 버전과 호환되는지 확인하려면 `magento/quality-patches` 패키지를 최신 버전으로 업데이트하고 [[!DNL Quality Patches Tool]에서 호환성을 확인합니다. 패치 검색 페이지](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ko). 패치 ID를 검색 키워드로 사용하여 패치를 찾습니다.

## 문제

[!DNL GraphQL]개의 장바구니 쿼리가 가상 견적에 대해 잘못된 할인 금액을 반환하여 부적격 가상 제품에 할인을 적용합니다.

<u>재현 단계</u>:

1. 두 개의 가상 제품 만들기:
   1. `virtual1` SKU
   1. `virtual2` SKU
1. 장바구니 가격 규칙을 만듭니다.
   1. 쿠폰 없이 자동으로 적용되도록 설정합니다.
   1. 작업:
   1. 제품 가격 할인의 백분율 - 20%.
   1. SKU가 `virtual2`인 장바구니 항목에만 적용됩니다.
1. Storefront를 통해 새로운 빈 견적에 제품 추가:
   1. 먼저 `virtual1` SKU를 추가하십시오.
   1. `virtual2` SKU를 초 단위로 추가합니다.
1. 다음 [!DNL GraphQL] 쿼리를 사용하여 장바구니 세부 정보를 가져옵니다.

   ```
   {
     cart(cart_id: "cart_id") {
       email
       itemsV2 {
         total_count
         items {
           uid
           prices {
             total_item_discount {
               value
             }
             price {
               value
             }
             discounts {
               label
               amount {
                 value
               }
             }
           }
           product {
             name
             sku
           }
           quantity
         },
         page_info {
             page_size
             current_page
             total_pages
         }
       }
       applied_coupons {
         code
       }
       prices {
         discounts {
           amount {
             value
           }
           applied_to
           label
         }
         grand_total {
           value
         }
       }
     }
   }
   ```

1. **[!UICONTROL cart]** > **[!UICONTROL items]** > **[!UICONTROL prices]** > **[!UICONTROL discounts]**&#x200B;을(를) 확인합니다.

<u>예상 결과</u>:

* `virtual1` SKU에 [!DNL GraphQL]에 나열된 할인이 없습니다.
* `virtual2` SKU에 [!DNL GraphQL]에 나열된 할인이 있습니다.

<u>실제 결과</u>:

* `virtual1` 및 `virtual2`이(가) [!DNL GraphQL] 응답에서 동일한 할인 금액을 잘못 표시합니다.
