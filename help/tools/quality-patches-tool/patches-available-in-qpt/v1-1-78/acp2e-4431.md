# ACP2E-4431: 색인 재지정 프로세스 중에 대상 규칙과 일치하는 [!UICONTROL Related Products]이(가) 삭제됩니다.

ACP2E-4431 패치는 색인 재지정 프로세스 중에 대상 규칙과 일치하는 **[!UICONTROL Related Products]**&#x200B;이(가) 삭제되는 문제를 해결합니다. 이 패치는 [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78이 설치된 경우에 사용할 수 있습니다. 패치 ID는 ACP2E-4431입니다. 이 문제는 Adobe Commerce 2.4.9에서 수정됩니다.

## 영향을 받는 제품 및 버전

**Adobe Commerce 버전에 대한 패치가 만들어졌습니다.**

* Adobe Commerce(모든 배포 방법) 2.4.7-p4

**Adobe Commerce 버전과 호환:**

* Adobe Commerce(모든 배포 방법) 2.4.4 - 2.4.8-p4

>[!NOTE]
>
>새 [!DNL Quality Patches Tool] 릴리스가 있는 다른 버전에 패치를 적용할 수 있습니다. 패치가 Adobe Commerce 버전과 호환되는지 확인하려면 `magento/quality-patches` 패키지를 최신 버전으로 업데이트하고 [[!DNL Quality Patches Tool]에서 호환성을 확인합니다. 패치 검색 페이지](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=ko). 패치 ID를 검색 키워드로 사용하여 패치를 찾습니다.

## 문제

색인 재지정 프로세스 중에 대상 규칙과 일치하는 **[!UICONTROL Related Products]**&#x200B;이(가) 삭제됩니다.

<u>재현 단계</u>:

1. 10000 이상의 제품을 만듭니다.
1. 5000개 이상의 카테고리에 10000 이상의 제품을 할당합니다.
1. 100개 이상의 관련 제품 규칙을 만듭니다.
1. **[!UICONTROL Product Category]** 사용과 일치하도록 **[!UICONTROL Products]**&#x200B;이(가) *X*&#x200B;인 규칙 조건을 구성하십시오.
1. **[!UICONTROL Product Category]**&#x200B;을(를) 사용하여 표시하도록 **[!UICONTROL Products]**&#x200B;을(를) 구성하는 것은 **[!UICONTROL Constant Value]** *Y*&#x200B;입니다.
1. 이 GraphQL 쿼리를 실행하여 제품에 대한 `related_products`을(를) 검색합니다.

   ```graphql
   {
       products(filter: { url_key: { eq: "simple-product-491" } }) {
           items {
               sku
               related_products {
                   uid
                   sku
                   name
                   url_key
                   stock_status
                   price_range {
                       minimum_price {
                           regular_price {
                              value
                           }
                           final_price {
                               value
                           }
                       }
                   }
               }
           }
       }
   }
   ```

1. `targetrule_product_rule` 인덱스에 대한 큰 백로그를 만듭니다.
1. `targetrule_rule_product` 인덱스에 대한 큰 백로그를 만듭니다.
1. `targetrule_rule_product` 인덱서를 실행합니다.
1. `targetrule_product_rule` 인덱서를 실행합니다.

<u>예상 결과</u>:

`related_products` 목록은 항상 규칙 조건과 일치하는 제품으로 채워집니다.

<u>실제 결과</u>:

인덱서가 실행되는 동안 `related_products` 목록에서 빈 배열을 반환합니다.
