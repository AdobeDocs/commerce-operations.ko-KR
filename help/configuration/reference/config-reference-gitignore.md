# .gitignore 참조

Magento Open Source에 기본 포함 `.gitignore` 파일. 자세한 내용은 [최신 상거래 `.gitignore`](https://raw.githubusercontent.com/magento/magento2/2.4/.gitignore) 파일. 에 있는 파일을 추가해야 하는 경우 `.gitignore` 목록, `-f` 커밋 준비 시 옵션:

```bash
git add <path/filename> -f
```
