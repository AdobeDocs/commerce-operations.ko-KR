# 서버 로그인

1. 서버에 로그인.
1. [파일 시스템 소유자](../installation/prerequisites/file-system/overview.md)(으)로 전환합니다.
1. 응용 프로그램을 복제한 디렉토리로 변경합니다. For example,

   ```shell
   cd /var/www/magento2
   ```
