# 서버 로그인

1. 서버에 로그인합니다.
1. 로 전환 [파일 시스템 소유자](https://devdocs.magento.com/guides/v2.4/install-gde/prereq/file-sys-perms-over.html).
1. 응용 프로그램을 복제한 디렉토리로 변경합니다. For example,

   ```bash
   cd /var/www/magento2
   ```
